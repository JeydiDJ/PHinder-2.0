# PHinder 2.0

PHinder is a Vue-based capstone project that has been rebuilt and refactored with the purpose of aiding the public regarding missing person cases in the Philippines through the website's built-in map pinning feature.

## Tech Stack
- Vue 3
- Vite
- Vue Router
