# PHinder-2.0
A restructure and redesign of the original PHinder project for the sole purpose of aiding the public search for missing persons with the use of the website's location pinning feature

## Built With
- Vue 3
- Vite
- Vue Router
- Pinia / Vuex
- Axios

## Features
- User authentication
- Search and filtering
- Responsive UI
- Dashboard view

## Installation
```bash
npm install
npm run dev
